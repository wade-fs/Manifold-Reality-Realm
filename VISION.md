# FSE World Design Vision

Flow State Engine 並不是為了重現某一本小說，也不是為了製作另一個 MMORPG。

我們想建立的是**一個真正存在的世界**。

在這個世界裡：

* 武功不是技能樹，而是對身體、呼吸、力與時機的理解。
* 修仙不是等級，而是世界承認你已經理解天地運行的規律。
* 神佛不是高等 NPC，而是世界規律的人格化。
* 門派不是職業，而是一群擁有共同世界觀與修行方法的人。
* 冒險不是完成任務，而是不斷發現未知、修正認知、理解世界。

玩家不需要被告知答案。

世界本身，就是老師。

NPC 可以誤導你。
古籍可能殘缺。
傳說未必可信。
甚至神明也只代表祂所理解的一部分世界。

真正的修行，不是累積經驗值，而是不斷觀察、推理、驗證，直到形成自己的理解。

每一次新的理解，都讓世界展現新的面貌。

因此，FSE 的核心循環不是：

**戰鬥 → 升級 → 裝備**

而是：

**生活 → 觀察 → 理解 → 驗證 → 成長**

我們希望創造的，不是一個等待玩家攻略的遊戲，而是一個值得玩家花上數百小時去生活、探索與理解的世界。

如果玩家離開多年後，仍然記得某座山、某位老人、某段旅程，以及某一天突然理解世界運作方式的那一刻，那就是 FSE 想創造的體驗。

# My Dream: A Living Chinese Mythology

我想建立的，不是《蜀山劍俠傳》，也不是《西遊記》，更不是另一個 MMORPG。

我想建立的是一個完整、連續、自洽的中國神話宇宙。

在這個宇宙裡：

* 神話不是故事，而是歷史。
* 神仙不是職業，而是對世界有不同理解的人。
* 妖不是怪物，而是不同生命演化出的文明。
* 門派不是技能樹，而是不同的世界觀。
* 武功不是技能，而是對身體與天地規律的理解。
* 修仙不是升級，而是世界承認你的理解。
* 法寶不是裝備，而是文明留下的知識結晶。
* 天庭不是政府，而是維護天地秩序的組織。
* 地府不是地獄，而是生命、記憶與輪迴的管理體系。
* 人間不是新手村，而是所有文明交會之地。

所有傳說，都可以同時存在。

《山海經》、《封神》、《西遊》、《聊齋》、《蜀山》……不是彼此衝突，而是不同時代、不同文明、不同立場所留下的紀錄。

玩家不只是經歷故事。

玩家將親自探索這個世界，逐步理解它，最後成為這段文明歷史的一部分。

我希望有一天，人們不是說：

「這是一款修仙遊戲。」

而是說：

「這是一個真正存在的神話世界。」
