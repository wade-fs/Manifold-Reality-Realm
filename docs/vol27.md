# 《修仙志怪錄》番外卷一 章節綱要

**卷名**：補一口井的四種方式
**性質**：番外篇（第六幕）‧第一卷
**章節**：番外第 1～4 章（不計入正文六百章）
**時間線**：第二幕至第三幕之間（對應正文約第25卷之後，第26卷之前的空檔）
**本卷主線**：涸田坳古井乾涸，四教修士各出手術，把一件微小的地脈問題，以完全合理的方式演化成無法靠近的小型時空錯亂秘境。陳進路過，插了一塊木板，說了五個字，問題就這樣了。

## 核心設定

**涸田坳**：洪崖縣外三里的小村落，五十幾戶人家，有一口存在了幾代人的古井。井乾涸後，村民開始做怪夢——夢裡都有水，但沒有辦法喝。

**四教失敗模式**（本卷核心笑點）：
- 遊玄教：「先做再說」→ 做了之後引來不可預測的副作用
- 觀虛教：「先分析再說」→ 分析完美，把水淨化到凡人喝了會死的程度
- 夢玄教：「讓認知先對了，現實自然對」→ 現實層沒有跟上
- 忘道教：「不想了，扔只鞋」→ 引發三種規則的共振災難

**終局機制**：四個歪歪扭扭的凡人文字「危險，勿近」，帶著一種最樸素的日常語義重力，讓四教所有的規則系統同時找到了退場的理由，以極其安靜的方式散了。沒有人能解釋這為什麼有效。陳進說不知道。

## 番外第 1 章　那口井空著

**提示**：用三分之一的篇幅建立涸田坳的日常煙火氣，再帶入「水不見了」的怪異。四個人的登場方式各自帶出性格——不要用旁白介紹，用他們到達的方式介紹。

**志怪核心**：一口乾涸的古井，村民做怪夢（夢裡有水但喝不到），這是地脈的微弱偏移，不難解決。

**開場**：寫涸田坳的日子——石子街、老槐樹、夏天的蟬聲和沒水的井口空著的聲音。村正跑了好幾個地方找人，最後找來四個修士，但他們不是一起來的。

**四人登場**：
- 齊三最早，申時初刻，背藥筐拄竹杖，從官道那邊過來，先去喝水聊天，聽完說「這種事不難，我見過」
- 白衍第二，路過，認為有責任確認這件異象的性質是否在可接受擾動範圍之內，換了乾淨外袍走到井邊，蹲下看很久
- 月素踩著葉子飄下來，在井邊繞一圈看天看地，說這口井的夢很淺，剛開始做、還沒想好要夢什麼
- 醉老被人搖醒，翻個身不動，最後月素在他耳邊說了什麼，他才站起來，搖搖晃晃跟著走

**齊三動手**：從附近撿石頭折樹枝，擺了一個「行道聚水陣」（簡陋但有效），加一把清熱草藥撒進去，井底開始冒出帶泥腥氣的水汽。他拍手說：「不難。」

**白衍的分析**：判定為「含有三種未標記雜質與一個無法歸類的意識殘留的錯誤輸入」，對齊三說那個陣的東南角少了一個固定點。

**收尾**：白衍說大概不算確認，月素說他們說的對，不代表井覺得他們說的對，醉老拔根草放嘴裡嚼著，說補什麼井，渴了去別的地方打水不就行了。

**留下的懸念**：齊三那個陣確實生效了——但那個意識殘留，白衍說的那個，到底是什麼？

- 障: 「先做再說」的人，在「先分析再說」的人旁邊動手，結果是兩個都說對了，但事情更複雜了
- 章回主角: 齊三（動作主角）、白衍（診斷主角）
- 收穫: 讀者認識了齊三和白衍最典型的反應方式
- 留下的因果: 那汪帶泥腥氣的水汽，混著一個無法歸類的意識殘留，懸在井口
- 作者提醒: 這一章要有煙火氣，讓涸田坳的人是真實的，不是道具

## 番外第 2 章　開始往壞處發展

**提示**：白衍施法，月素施法。節奏是「每一個人的施法都很合理，但組合起來的效果完全無法預測」。用白衍和月素的對話帶出兩種世界觀的根本語義衝突。醉老在旁邊靜靜地喝他葫蘆裡的東西，沒有說話。

**白衍動手**：拿出「九轉淨水無暇丹」，投入井中，把那汪渾濁的水淨化成完全符合五行標準定義的純淨靈液。淨化後，那口井的水純淨到：泥腥氣消失、腐草的濕意消失、甚至那說不清楚的生機也一起消失了。村子裡一個孩子來打水，白衍把她攔住，說規格超過人體承受上限，孩子說那和沒水有什麼差別。

**月素動手**：「太陰迷離幻陣」，讓井口成為一面心理之鏡，讓靠近的人在水面看見自己關於水的最深的記憶，讓那口太乾淨的井重新有了人的溫度。同時丟進一顆「醉夢引」——喝了不會死，只是會在夢中過完一生，想通某件事才醒。她說：「這樣可以用了。」

**白衍和月素的辯論**：
- 白衍說：你的幻陣篡改認知，讓人以為那口水是正常的，但它實際上不符合飲用標準
- 月素說：以為可以喝，就可以喝了
- 白衍說：喝完會怎樣？
- 月素說：會做一個夢。大概。
- 白衍說：大概不是一個精確的答案。
- 月素說：夢本來就不精確，精確的夢是最無聊的夢。

**醉老動手**（沒有任何預兆）：把腳上的鞋脫下來，往井裡扔了，說了一句話（齊三後來試圖回憶，記不清楚，大概意思是「這口井太正經了，就這樣吧」），回到石頭上繼續睡覺。

**鞋子引發的效應**：那只破鞋在純淨靈液和夢境幻陣之間，找到了一個奇怪的平衡點，懸停，開始旋轉，帶動靈力流場，旋轉方向恰好和白衍的五行鎖陣相反，和月素的幻陣頻率形成共振。

**地底的聲音**：三種規則開始互相咬合，地底傳來一聲低沉的、像是什麼東西清醒過來的聲音。

- 障: 每個人的出手都是基於完整的判斷，但沒有人問過別人打算做什麼
- 章回主角: 白衍（施法主角）、月素（施法主角）、醉老（引爆者）
- 收穫: 三種規則的正反饋迴路啟動
- 留下的因果: 那口井現在是三種規則耦合的交界點
- 作者提醒: 醉老扔鞋這個動作，要寫得像一個完全隨性的、甚至是懶洋洋的動作，不要讓它看起來像是在「施展術法」

## 番外第 3 章　完全失控

**提示**：這一章是番外卷一的高潮。從「稍微複雜」加速到「完全無法收場」的過程，要寫得既荒誕又有一種殘酷的邏輯一致性。每一件更壞的事，都直接從前一件合理的嘗試裡長出來。

**失控的過程**：
- 齊三試圖用引水訣把那個從三規則耦合裡冒出來的東西引走，但引水訣的語義在夢玄教幻陣裡被折射了，他拉出的不是水，而是某個村民昨晚的夢境殘影——一條河，一座橋，橋上站著一個不認識的孩子
- 白衍試圖用空間切割術把那個殘影移除，但夢境沒有精確座標，切割每次落空，在空間裡留下了一道道針眼大小的縫隙，讓井口上方的空氣帶上了輕微的折射率
- 一隻路過的麻雀被切割術的殘留鎖定，切掉兩根羽毛，羽毛落進幻陣，沉進忘道教的鞋子攪起的漩渦，消失。那隻麻雀在五十步外的樹枝上突然陷入深度睡眠
- 月素看見空間縫隙說「有趣」，試圖用因果鏡射術把縫隙的開口和這口井最初的記憶連接起來——但那個最初記憶已經被白衍的九轉淨水無暇丹淨化殆盡，月素的鏡射術找到的，是那口井在「九轉淨水無暇丹」世界觀裡的理想形態：一個完全沒有顏色、沒有溫度、沒有味道的純粹水源
- 那個純粹水源被拉出來，覆蓋在一切之上，整個井口上方出現一個直徑三丈、完全靜止的白色空域
- 一隻飛鳥路過，被切割術殘留削掉三根羽毛，被幻陣捲走視覺，被忘道鞋的漩渦拂了一下，在物理意義上變成了一隻杯子，掉在地上，裡面有半杯水，那個水是某個夏天一個女孩打水的記憶

**三教死鎖**：
- 白衍說：誰先撤，爆炸半徑一里半到兩里
- 齊三說：那月素先撤？白衍分析：月素的幻陣現在是緩衝層，撤了就是白色空域直接和地脈接觸
- 月素說：可能是空間折疊，可能是時間分裂，可能是什麼都沒有，可能我說的全是錯的（其中有理論支撐的有一個，但她記不清楚是哪個）
- 醉老繼續睡覺，呼嚕聲在寂靜裡顯得格外清晰

齊三撿起那隻變成杯子的鳥，看了很久，說：「這真的不在計劃之內。」白衍說：「我說過了，你的陣東南角少了一個固定點。」

- 障: 三種正確的分析，在同一個空間，形成了一個誰都無法後退的囚徒困境
- 章回主角: 齊三（失控主角）、白衍（分析囚境）、月素（加速失控者）
- 收穫: 白色空域完全展開，讀者知道這件事已經徹底失控
- 留下的因果: 那隻變成杯子的鳥，那半杯水，那個女孩打水的記憶
- 作者提醒: 這一章每一件壞事都要有因果邏輯，不是「隨便出問題」，而是「每一個問題都能追溯到上一章誰做了什麼」

## 番外第 4 章　一塊木板

**提示**：陳進出現。節奏要突然慢下來——不是因為他帶來了答案，而是因為他帶來了一種「也許這件事不需要繼續」的氛圍。他的每句話都很短，沒有一句是在解釋，只是在確認事實。木板插進去的那一刻，要寫得極輕，像一塊石頭放進水裡——不是爆炸，是沉。

**陳進登場**：穿破舊灰布道袍，背藥筐，牽一條黃狗，從田埂那邊走過來，鞋底沾著不同地方的泥。黃狗比他快半步，嗅一圈，退到老頭腳邊趴下。齊三第一個認出那根鐵皮箍的木杖。

**陳進的問法**：
- 「這口井，本來是怎麼了？」（乾涸了）
- 「現在怎麼了？」（我們讓它變成這樣的）
- 「能撤嗎？」（撤不了）
- 「村民呢？」（在東邊廟裡等消息）

**陳進的動作**：往藥筐裡翻，拿出一塊包草藥的破木板，用炭筆寫了四個歪字：**「危險，勿近」**。插進井邊的泥土裡，拍手上的灰。說：「這口井廢了，別讓村民靠近。去跟村正說，往南邊再挖一口，地方我看過了，不難。」

**術法如何散的**：
- 月素問：「你看懂了嗎？這口井現在是什麼狀態？」陳進說：「沒有，不知道。」
- 月素問：「你說不知道，是真的不知道嗎？」陳進說：「是真的。」
- 月素說：「好。」緩緩鬆開法訣，夢境的幻陣像霧一樣散了
- 白衍說：「等等，你沒有按照正確的順序——」月素說：「因果自己知道順序。」
- 白衍收起三百六十面陣旗，白色空域開始分解，極慢地變薄、變透明、消失
- 空間縫隙一道一道閉合

**醉老醒來**：日落的時候，問「我的鞋呢」，聽說在井底，說「算了，那只鞋走的路夠多了」，走了。走到槐樹邊，轉身說：「新井選在槐樹南邊，別選石頭底下的水，石頭底下的水太硬，老人喝了腿疼。」說完沒等人回應，走了。

**收尾**：
- 齊三站在廢井邊看著那塊木板，把陳進說的話又想了一遍：「你弄了之後你才知道，弄了也沒用。」他說不清楚那是不是什麼道理，但他把它記下來了。
- 白衍：「忘道教的人，忘的是所有不必要的部分，必要的部分他們沒有忘。」月素說：「說話和意思，不是同一件事。」白衍說：「你這句話同樣不精確。」月素說：「我知道。」

- 障: 四種正確的世界模型，同時作用於一個不需要被建模的問題
- 章回主角: 陳進（破局，以不破局的方式）
- 收穫: 一口廢井，一塊木板，新井在南邊，老人腿不疼
- 留下的因果: 那塊木板的四個歪字為什麼能解鎖四教術法——陳進說不知道，是真的不知道
- 作者提醒: 陳進在這一章不能有任何「我知道問題所在」的神情，他就是一個路過的老頭，看了一眼，覺得最省事的辦法是插個牌子
