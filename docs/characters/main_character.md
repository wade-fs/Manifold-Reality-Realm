# 主角設定：陳進

## 人物定位

**姓名**：陳進

陳進年約六十歲，入門五十餘年，是門中公認的「少年天才」。在凡人眼中，六十歲已近暮年；在修仙界，卻仍只是漫長修行的起點。

他修行速度驚人，年未六十便踏入元嬰，因此符合門派「行道院」的資格，被任命為**行道長老**，並兼任**傳功院教導**。

宗門長輩對他的評價始終只有一句：
> 少年得志，心性尚需磨鍊。

因此，他真正的修行，不是在山門，而是在天下。

---

## 人物核心：不知道的力量

很多長老，一輩子守著洞府。
很多峰主，一輩子守著山門。
只有陳進，一直在路上。

陳進下山，不是因為他懂了什麼，而是因為他**什麼都不懂**。
他不是帶著真理來拯救世界的聖人，而是一個被世界拋出的無數詭異現象弄得暈頭轉向的旅人。

當別人問他：「你為何出山？」
他的回答是：**「不知道。」**
這不是高深莫測的謙虛，而是他真的不知道。

他最大的本事，就是**承認自己的無知**。面對光怪陸離的志怪現象、扭曲的規則、荒誕的神佛，他不會急著拔劍，也不會講述大道。他只是觀察、發問、甚至犯錯。

## 人物鮮活細節（言行與習慣）

- **標誌性神態**：
  - **嘆息**：遇到無法理解的荒誕與苦難時，他最常做的動作是「嘆氣」。這不是悲天憫人，而是面對荒謬世界的無力感。
  - **溫和與迷惘**：他極少發怒，面對別人的質疑，他通常帶著一絲迷惘的微笑，彷彿在說「其實我也搞不清楚狀況」。
- **極致的煙火氣**：他能毫無違和感地坐在火堆旁、茶棚裡，與凡人聊家常。他看起來就像個普通的行腳大夫或落魄老頭。
- **戰鬥哲學（不爭與隨緣）**：
  - 面對敵人，他極少主動祭出殺招。他更習慣順應現象，用極度反常規或凡俗的方式（如遞一杯水、吹滅一盞燈）去觸碰因果的線頭。
- **「陳進很舊」**：
  - 見過他的人，說法都不一樣——像翻很多次的經書，像曬退色的門神，像一把用了三百年的鋤頭，像一塊很老的石頭。每個人的說法不一樣，但方向是同一個。永遠不解釋為什麼。

## 隨身物品（名詞延遲命名原則）

陳進身上的東西，一開始都沒有威風的名字，他自己也不完全知道怎麼用。

| 物件 | 初始狀態 | 故事中的演變 |
|:---|:---|:---|
| **破木杖** | 一根箍了鐵皮的普通木拐杖，用來探路 | 識別特徵，多人見過認出 |
| **破銅鏡** | 佈滿銅綠的破鏡子，偶爾照出詭異幻象 | 第21卷摔裂，修補後照人微歪；偶爾歪的角度照出尋常角度看不見的東西 |
| **未開刃的鐵劍** | 一把沒有鋒芒的鈍劍 | 基本不用，象徵不以殺伐為目的 |
| **藥筐（竹篋）** | 裝了幾卷殘缺古籍與水囊 | 古籍空白頁持續減少；第42卷後以草紙補充；草藥不斷更替 |
| **土地公泥塑神像** | 第1卷取得 | 備而未用，同行至第三幕 |
| **通行無礙銅錢（×2）** | 一枚自己的，一枚黃嬸借而未還 | 兩枚都帶著 |
| **常三的廢圖** | 第26卷第7天，常三送的最後一張圖 | 折了很多次，早已看不清楚，但一直帶著，第60卷末還在藥筐裡 |
| **聚義堂木牌** | 第23卷取得的「榮譽顧問」木牌 | 夾在古籍裡，無實際法力 |

> **古籍特別說明**：古籍的空白頁是有成本的東西。每次陳進撕出一頁，空白就少一頁。第42卷發現只剩最後一頁時，他沒有寫字，合上，後來在集市上買了草紙夾進去「先湊著用」。古籍裡有幾頁從未裁開的空白，第44卷在問者那個地方，那幾頁自己開了，裡面有幾個字，極古老，陳進認識但說不清楚意思。第72卷他再次讀了，理解了一個字，合上，繼續走。

## 宗門與行道院

- **宗門祖訓**：> **未行萬里，不得論道。**
- **行道長老職責**：巡視附屬宗門、處理地方怪異、探查新秘境。行道長老不是宗門最強的人，而是宗門看過世界最多、心中充滿最多疑問的人。
- **陳進的最終定位**：遊玄教**入世玄心派**行道長老。他以「問」為刃，手持殘劍，身背布袋，入世歷劫，降伏心魔，是遊玄教中「少年得志、心性需磨」的最典型代表。

## 境界與心境的演變（全書五幕弧線）

### 第一幕：行道初章（第1～10卷）
- **境界**：元嬰初期
- **外表**：走得穩的老頭，靈壓隱而不散
- **「不知道」的性質**：帶著謙虛習慣，尚未真正放手
- **問法**：從「我能做什麼」，走到「這裡需不需要我做什麼」

### 第二幕：深入山海（第11～25卷）
- **境界**：元嬰後期（第25卷末突破出竅）
- **外表**：靈壓幾乎完全收斂，普通人見到不會有壓迫感，就是「走了很多路的老頭」
- **「不知道」的性質**：從習慣性謙遜，完成轉化為真實的放手
- **問法**：兩件事都對，走哪條路？

### 第三幕：問鼎之前（第26～45卷）
- **境界**：出竅→化神→分神初期
- **外表**：徹底凡人化；偶爾被石頭絆倒；買東西會還價；在廟裡打瞌睡。修行界的人開始刻意迴避，說不清楚為什麼。
- **問法**：如果世界的運轉方式本身就是一種障，怎麼辦？
- **重要事件**：第36→40卷，第一次真正的失敗——他做了一個他認為正確的選擇，那個選擇是錯的，有真實的傷害。他不自責，不崩潰，但也沒有原諒自己，因為這件事不在原不原諒的框架裡，它就是因果，它就在那裡。

### 第四幕：合道之路（第46～60卷）
- **境界**：分神→合體→煉虛初期
- **外表**：看起來年紀不定——有人說三十歲，有人說一百歲，有人說「像一塊很老的石頭」；連修行界同道有時也認不出他；被當作路邊乞討的老頭。
- **問法**：當一個人能夠做任何事，他為什麼選擇只做最小的那件？
- **行道方式**：一杯水、一句話、吹滅一盞燈、在河邊坐了一個下午、替人搬了半天貨。不是刻意低調，是他就是這樣了。
- **破局特徵**：志怪規模越大，他的動作越小。讀者看不見他「刻意克制」，只能看見他就是這樣做的，像喝水一樣自然。

### 第五幕：問鼎與歸（第61～75卷）
- **境界**：問鼎→寂滅邊緣（真仙未至，但已不重要）
- **外表**：不重要了。他在哪裡，哪裡就是一個可以停的地方。有時候在市集，有時候在路邊，有時候開了一家茶館幾個月，然後又走了。
- **存在的方式**：他在那個地方，那件事就有了一點不同的走向。說不清楚因果在哪裡。不解釋。
- **問法**：什麼都不再是問題的時候，他還在哪裡？
- **結尾**：那口老井，那個傍晚，那個他說不清楚名字的問題。他蹲在井沿，看了很久，然後站起來，扛起藥筐，往另一個方向走去。沒有結局，只有繼續。

## 褪去仙氣，回歸凡俗

在經歷了幾百年的遊歷、見證了無數天道與規則的崩壞後，陳進的修為雖然達到了頂尖，但他的外在表現走向了極致的反面——**徹底的凡人化**。

- **與法則共生**：他不再試圖對抗或掌控法則，而是與之共存。法則變成了他骨骼裡的鈣質。
- **放棄仙人姿態**：他主動散去靈壓，看起來就是一個毫無修為的普通老頭。他會感覺冷、會覺得餓，甚至會被石頭絆倒。唯有褪去仙人的無敵保護殼，才能真正觸碰人間的真實。
- **不說教的溫柔**：到了後期，他連「發問」都少了。他開了一家小酒館，或者開了幾個月茶館，或者牽著一條黃狗在鬧市裡溜達（後來那條狗自己跑走了），或者坐在路邊替人看診。他用最平常的善意和不可思議的巧合，化解世間的苦難。最高的大道，就在這最平凡的市集煙火之中。

## 行道原則（寫作者備忘）

- **破局方式**：永遠是「最小的凡俗動作」——列帳、說一句話、問一個問題、幫人修籬笆、點兩根香、在每一個桌上放一盞燈。
- **說話方式**：極少主動解釋，說話前習慣確認事實，說話後不等對方反應就走。話越來越短，第五幕裡有時一卷只說幾句話。
- **對神的態度**：不崇拜，不輕視，對話時說話直接，把神當成一個可以溝通的存在，而不是一個需要討好或對抗的力量。
- **記錄習慣**：重要的事情，在古籍空白頁上用炭筆記下，格式極簡（地名+幾個字）。第8卷開始有記神的習慣，日後演變為行道重要工具。
- **「不知道」的使用**：「不知道」說出口，永遠是真實的，不是謙虛，不是迴避。這一點是問者和封印等存在判斷「算不算」的標準，也是全書最重要的單句。到了第五幕，這句話說得越來越少，不是因為他知道了，而是因為「不知道」和「知道」之間，那條線在他身上消失了。
