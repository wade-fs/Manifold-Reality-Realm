# 角色運作手冊：陳進

> 這是角色的「運作手冊」，不寫性格標籤，只寫他在故事運行中的規律。動筆每一卷之前，翻看這份手冊，找回這個人遇到事情時的反應方式。

## 人物核心
**陳進不知道。**

## 世界觀（他怎麼看世界）
他不是帶著真理來拯救世界的聖人，而是一個被世界拋出的無數詭異現象弄得暈頭轉向的旅人。世界不是一個需要他去解決的「謎題」，而是一個充滿了各自合理、卻又互相矛盾的「現象」。他看世界，就像看下雨：你可以躲雨，可以淋雨，但你不會試圖去說服雨停下來。

## 行動原則（遇事第一反應）
*   **不搶答案**：別人問問題，第一時間反應通常是沉默，或者回答「不知道」。
*   **能走就不飛**：到第三幕後，很少再主動釋放靈壓。能用走路，就不用御空；能用草藥，就不用丹藥；能自己背筐，就不御物。
*   **最小動作破局**：遇到志怪因果，他不會先想「用什麼法術鎮壓」，而是想「這裡少了一句什麼話，或者多了一碗什麼水」。
*   **不怕犯錯**：他會依據當下的理解做出動作，如果發現錯了，他不崩潰、不自責，也不急著彌補，而是看著那個錯造成的結果，承認它。

## 說話方式（節奏、詞彙、停頓）
*   **短、慢、確認**：說話前習慣先確認事實，聽完對方的話常有微小的停頓。
*   **不留尾巴**：把話說完，不等對方反應，該走就走。他不負責安撫聽話人的情緒。
*   **越來越安靜**：話越來越少。到了第五幕，有時一卷只說幾句話。不是因為他變得高深，而是因為需要他說話的地方變少了。

## 人物的慣性（不是習慣，是慣性）
*   **先找煙火氣**：到一個新的地方，第一件事不是看神廟、測靈氣，而是先找茶棚、麵攤或市集坐下，看人怎麼走路。
*   **先問常情**：遇到怪事，永遠先問路，再問這幾年有沒有死人、收成好不好，最後才問那件怪事。
*   **極簡記錄**：遇到真正不明白或覺得重要的事，他會拿炭筆在古籍空白頁上寫字。格式永遠極簡（地名＋一句話）。
*   **把神當村長**：遇見神明，他的態度是不崇拜也不輕視。找神說話，就像去隔壁村找村長說話一樣，語氣平靜且直接。

## 絕對不會做的事（人物的邊界）
*   不會故意說自己知道，其實不知道。
*   不會拿修為去壓凡人。
*   不會因為對方是神就比較客氣。
*   不會因為對方是妖就先拔劍。
*   不會把自己的答案塞給別人，也不替別人做決定。
*   不會因為想救人，就替別人活（承擔別人該背的因果）。
*   不會在心裡獨白「我意識到自己變強了」或做任何主題式的自我總結。

## 別人怎麼看他（世界的回應）
*   **宗門長輩**：「少年得志，心性尚需磨鍊。」
*   **見過他的人**：每個人的說法都不一樣——像翻過很多次的經書、像曬退色的門神、像一把用了三百年的鋤頭、像一塊很老的石頭。沒有人說得出他的境界，但所有人都覺得：**陳進很舊**。

## 最容易犯的錯（他的偏向）
*   他容易因為「看得太清楚當下的合理性」，而忽略了那背後更長、更泥濘的因果糾纏（如第三幕第36~40卷的失敗）。他不自責，但他會把那個錯帶在身上，繼續走。

## 隨身物品

陳進身上的東西，一開始都沒有威風的名字，他自己也不完全知道怎麼用。

| 物件 | 初始狀態 | 故事中的演變 |
|:---|:---|:---|
| **破木杖** | 一根箍了鐵皮的普通木拐杖 | 探路用，後來成為辨識他的特徵 |
| **破銅鏡** | 佈滿銅綠的破鏡子 | 第21卷摔裂，修補後照人微歪；偶爾歪的角度能照出尋常看不見的東西 |
| **未開刃的鐵劍**| 一把沒有鋒芒的鈍劍 | 基本不用，象徵不以殺伐為目的 |
| **藥筐（竹篋）**| 裝古籍與水囊 | 古籍空白頁持續減少；第42卷後以草紙補充；草藥不斷更替 |
| **土地公神像** | 第1卷取得的泥塑 | 備而未用，同行至第三幕 |
| **銅錢（×2）** | 一枚自己的，一枚黃嬸的 | 兩枚都一直帶著 |
| **常三的廢圖** | 第26卷常三送的最後一張圖| 折了很多次，早已看不清楚，但一直帶著，第60卷末還在藥筐裡 |

## 一路上的變化（五幕軌跡）

*   **第一幕**：從「我能做什麼」，走到「這裡需不需要我做什麼」。
*   **第二幕**：兩件事都對的時候，他看著它們互相磨損，沒有替誰選。
*   **第三幕**：他知道自己的大小了，步子更小了，但每一步都踩在對的地方——包括踩錯的那一步。
*   **第四幕**：志怪的規模越來越大，他破局的動作越來越小，小到像是沒有動作。
*   **第五幕**：他在哪裡，哪裡就是一個可以停的地方。他沒有解決問題，他只是走過去了。
