1. 主角：陳進（Chen Jin）
背景
出身南方偏遠山村的普通農家子弟，父親是獵戶兼佃農，母親早逝。從小幫忙農事，卻總在田間、霧中、山路上發呆觀察「不一樣的地方」——例如同一片田地，不同人走過後作物生長竟有細微差異。村塾先生教書時，他總問「為什麼」，被嘲笑為「問呆子」。15歲那年，村中大霧七日不散，他第一次走進霧裡，看見童年自己與未來模糊身影同時存在，從此踏上追問之路。
角色類型：漸進理解者（Adaptive Interpreter）→ 最終進化為穿越理解者（Cross-Reality Interpreter）→ 終極觀測者。
理解模型參數（隨卷動態進化）

Observation Model（觀測模型）：
初始：「眼見＋心疑」——世界應穩定，但任何細微不一致都值得記錄與追問。
後期：「多層同時觀測」——能同時看見多個實相層（人間、妖界映射、天庭規則、地府記憶）。
State Representation（狀態表示法）：
初始：「差異地圖」——用筆記本記錄「什麼與什麼不符」。
中期：「理解層疊圖」——將不同門派、神話版本視為可疊加的透明紙。
後期：「實相樹」——世界是一棵不斷分支的樹，他是維持主幹與分支張力的根節點。
Transformation Rules（轉換規則）：
核心能力：「借用－融合－維持」。
借用他人的理解模型來觀測現象。
融合矛盾理解而不立即崩壞。
後期能「維持張力」——讓互斥實相同時穩定存在。

Stability Constraint（穩定條件）：
必須「持續好奇與好問」。一旦停止追問，理解模型就會僵化。
最終穩定條件：無我承載——當他不再執著於「我是誰」時，世界即可自穩定。

性格驅動：好奇是燃料，好問是方法，好學是路徑。平庸出身讓他沒有門派包袱，能更自由地在不同理解間遊走。

2. 張固執（Zhang Guzhi）—— Rigid Interpreter 代表
背景
原為縣衙老書吏，50餘歲。一生整理檔案，深信「世間萬物皆有唯一正解」。第一卷時因地圖錯亂而崩潰，創立「定實宗」，主張以強力修正恢復單一實相。後成為天庭收斂派的合作者。
角色功能：衝突製造者、悲劇鏡像（映照陳進若走極端會如何）。
理解模型參數

Observation Model：嚴格「一致性檢測」——任何與記錄不符者皆為錯誤。
State Representation：單一「正史線」——世界是一本只能有一個版本的史書。
Transformation Rules：強制「修正與覆寫」——用文書、法令、規則強行統一現象。
Stability Constraint：必須維持「唯一性」。一旦承認多版本，即自我崩解（第五卷徹底消失）。


3. 青霧（Qing Wu）—— 妖界 Cross-Reality Interpreter
背景
出身山中妖霧邊緣的「半妖」——母親是人間女子，父親是早期妖界語言變異體。出生時身體半透明，語言常自動分岔。被村人視為不祥，後被陳進在霧中救下，成為最早旅伴。性格靈動、善變、帶點玩世不恭。
角色功能：提供非人類視角，幫助陳進理解妖界生成引擎。
理解模型參數

Observation Model：流變觀測——世界本質是持續變化的霧，固定形態只是暫時共識。
State Representation：語言生態網——萬物皆由可生長、分裂、融合的語言/符號構成。
Transformation Rules：敘述即改造——說出或描述某物，即改變其存在形態。
Stability Constraint：必須保持「流動性」。過度固定形態會讓她實相固化（接近死亡）。


4. 墨史（Mo Shi）—— 地府相關 Adaptive Interpreter
背景
前地府低階書吏，因整理多版本記憶檔案時出错，被罰轉生人間。保留部分記憶壓縮能力，性格沉穩、略帶書卷氣，常以「記錄者」自居。與陳進因共同整理異常記錄而結識。
角色功能：記憶與歷史維度的支柱，幫助處理坍縮危機。
理解模型參數

Observation Model：壓縮視角——只關注「可被儲存的核心模式」，忽略噪聲。
State Representation：記憶層疊庫——世界是多版本歷史的壓縮檔案。
Transformation Rules：選擇性重載——可提取某版本記憶注入現實，或壓縮衝突記憶。
Stability Constraint：必須「定期歸檔」。若記憶過載，會陷入永恆夢境。


5. 天樞子（Tian Shu Zi）—— 天庭前節點
背景
原天庭中階規則節點，因第一卷修正失敗而「墜凡」，化為遊方道人。外表清癯老者，語氣總帶系統性，習慣用「應當」「必須」說話。後分裂為容納派代表。
角色功能：體現天庭引擎的內部衝突與轉型。
理解模型參數

Observation Model：規範檢測——世界應符合預設約束集。
State Representation：規則網絡圖——萬物由因果、時間、命運等節點連結。
Transformation Rules：修正與鎖定——可局部重寫規則，但消耗「天庭權限」。
Stability Constraint：必須服從「整體穩定優先」。在第四卷後逐漸接受多版本，約束鬆動。


6. 無名（Wu Ming）—— 最終空觀測者
背景
第七卷才正式登場。無固定來歷，可能是在多次實相重啟中「被遺忘的殘片」聚合而成。外表如普通青年，眼神空洞卻澄澈。無門派、無記憶、無強烈慾望。
角色功能：最終催化劑，幫助陳進完成「觀測消散」。
理解模型參數

Observation Model：空觀——無預設立場，純粹接收所有輸入。
State Representation：無中心拓撲——世界無需核心即可存在。
Transformation Rules：不干涉——存在本身即是最大的轉化。
Stability Constraint：無執——沒有任何必須維持的條件，因此最穩定。


人物關係與群像弧線簡表

陳進：核心黏合劑，連接所有人。
陳進 vs 張固執：理解極端對立（好奇 vs 固執）。
陳進 + 青霧：互補成長（人間 vs 妖界）。
陳進 + 墨史：記錄與實踐的搭檔。
陳進 vs 天樞子：規則 vs 靈活的哲學對話。
陳進 + 無名：最終的「有 vs 無」融合。

所有人物的理解模型都可直接套入門派 = 實相算法公式，方便後續生成章節時快速調用。
