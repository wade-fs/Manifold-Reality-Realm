# 《多重實相界》主要角色檔案（main_characters.md）

## 陳進（主角）

**種族**：人族  
**年齡**：第一卷初期約18歲  
**性格**：好奇、好問、好學、心地善良，但有點優柔寡斷。  
**背景**：南方山村獵戶之子。出生時有紫氣東來異象，母親在當夜神秘消失。  
**能力**：  
- 擁有母親「陽」之碎片（紫氣）  
- 能感知多重實相  
- 逐步發展出「理解改變世界」的能力  
**成長弧**：  
- 初期：困惑、被動逃亡  
- 中期：主動遊歷、學習融合紫氣  
- 後期：成為能承載所有版本的「終極觀測者」  
**與郭芳的關係**：從互相看不順眼，到一路同行、互相扶持、最終成為心愛的伙伴。

---

## 郭芳（心愛伙伴）

**種族**：上古貓族妖修  
**年齡**：外表約20歲  
**性格**：獨立、倔強、行動力強、嘴硬心軟。常與陳進意見相反（陰陽對立），但關鍵時刻極具保護欲。  
**背景**：母親分裂出的「陰」之碎片化身。出身偏遠妖族山域，從小感覺與族人不同，常做關於混沌初開的夢。  
**能力**：  
- 擁有母親「陰」之碎片  
- 極強的直覺與感知力  
- 速度極快，擅長隱秘突襲與夜行  
- 可顯露貓耳與尾巴（力量全開時）  
**成長弧**：  
- 初期：高冷、警覺、獨來獨往  
- 中期：逐漸信任陳進，開始與他共同面對危機  
- 後期：與陳進的紫氣徹底共鳴，成為真正的心愛伙伴  
**與陳進的關係**：陰陽對立 → 互相理解 → 並肩同行 → 心靈相通。

## 其他重要角色

**張固執**  
- **身份**：定實宗核心人物  
- **性格**：極端執著、堅持單一真實、原則性強  
- **角色功能**：陳進的鏡像與對立面，代表「一」的極端。  
- **與主角關係**：初期為潛在敵人，後期可能成為亦敵亦友的存在。

**墨史**  
- **身份**：地府相關人物（幽冥書吏）  
- **性格**：沉默、博學、帶點蒼涼  
- **角色功能**：記憶與歷史的守護者，提供關鍵情報與世界觀解釋。  
- **與主角關係**：中立盟友，常在關鍵時刻出現。

**青霧**  
- **身份**：神秘的語言生態師（妖族）  
- **性格**：悠然、睿智、帶點玩世不恭  
- **角色功能**：提供神話與妖界視角，偶爾幫助主角。  
- **與主角關係**：亦師亦友，偶爾出現指點迷津。

**天樞子**  
- **身份**：天庭使者（收斂派）  
- **性格**：冷峻、嚴苛、忠於秩序  
- **角色功能**：主要反派之一，代表天庭的強硬勢力。  
- **與主角關係**：主要追殺者之一。

**無名**  
- **身份**：混沌殘靈的化身  
- **性格**：神秘、中性、超然  
- **角色功能**：提供關於母親與最初一息的關鍵資訊。  
- **與主角關係**：中立存在，偶爾以謎語形式出現。
