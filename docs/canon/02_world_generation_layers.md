# 世界生成分層系統（World Generation Layers）

## 核心目的

本文件定義：

「《多重實相界》的世界是如何被生成的」

不是設定世界長什麼樣，而是定義：

世界是怎麼被『組出來』的。

--------------------------------------------------

## 一、三層世界結構

世界不是單層存在，而是三個同時運作的生成層。

--------------------------------------------------

## ❶ 神話層（Myth Layer）

### 功能

決定「世界為什麼存在」

這一層不是地圖，也不是歷史，而是：

- 概念如何變成現實
- 世界如何被理解
- 規則如何被改寫

### 內容類型

- 混沌之母
- 紫氣碎片
- 三界本質
- 因果法則
- 理解即變化

### 特徵

- 不隨時間線改變
- 是所有事件的根源
- 直接影響敘事規則

--------------------------------------------------

## ❷ 敘事層（Narrative Layer）

### 功能

決定「故事如何發生」

這一層是：

- 事件
- 角色行動
- 衝突與解決
- 旅程結構

### 內容類型

- 陳進的旅程
- 郭芳的選擇
- 神與妖的衝突
- 各卷劇情
- 紫氣碎片收集

### 特徵

- 可變
- 可重寫
- 是故事主體

--------------------------------------------------

## ❸ 填充層（Filler Layer）

### 功能

決定「世界如何看起來像真實的」

這一層不影響主線，但提供：

- 密度
- 現實感
- 存在感

### 內容類型

- 村民日常
- 市集交易
- 城鎮運作
- 一般妖怪活動
- 非關鍵事件

### 特徵

- 可隨機生成
- 可重複使用
- 不影響主線因果

--------------------------------------------------

## 二、三層之間的關係

### 神話層 → 決定規則
### 敘事層 → 使用規則
### 填充層 → 偽裝真實

--------------------------------------------------

## 三、關鍵設計原則

### 1. 神話層不可被填充層污染

背景角色不能影響神話理解。

例如：

村民不能改變「紫氣是什麼」。

--------------------------------------------------

### 2. 敘事層可調用填充層，但不能反向

主角可以進入村莊世界，但村莊不能改寫主線法則。

--------------------------------------------------

### 3. 神話層只由「允入角色」觸發變動

只有符合「人物允入條件」的角色才可以：

- 改變世界理解
- 觸發世界重構
- 影響神話層結構

--------------------------------------------------

## 四、世界生成流程（標準化）

每一個章節或區域生成，必須依序：

STEP 1：
從神話層確定「這裡的理解主題」

STEP 2：
在敘事層生成「可發生事件」

STEP 3：
在填充層補足「現實細節」

--------------------------------------------------

## 五、設計意圖（作者備註）

這個系統的核心不是分類世界，而是：

讓世界永遠保持「神話感 + 真實感」的共存。

如果只有神話層 → 變成哲學抽象

如果只有敘事層 → 變成一般小說

如果只有填充層 → 變成背景世界

三層同時存在 → 才是「多重實相界」
--------------------------------------------------
