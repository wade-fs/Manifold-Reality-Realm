# 神話記憶系統（Myth Memory System）

## 定位

本文件屬 canon 正典規則。

它定義：

「世界如何記住自己曾經被不同方式理解過。」

--------------------------------------------------

## 一、記憶不是記錄，而是殘留理解

在本世界中：

- 記憶 ≠ 事實保存
- 記憶 = 被保留下來的理解版本

因此同一事件可能同時存在：

- 物理事實
- 神話版本
- 錯誤版本
- 遺失版本

--------------------------------------------------

## 二、三層記憶結構

世界記憶由三層構成：

--------------------------------------------------

### 1. 表層記憶（Surface Memory）

- 人類與低階存在可直接存取
- 會隨時間變動
- 容易被合理化或遺忘

例：
「某村曾經有神像」

--------------------------------------------------

### 2. 紫氣記憶（Qi Memory）

- 由理解殘留構成
- 不依賴文字或歷史
- 以現象方式存在

例：
- 同一地點總會出現類似事件
- 某種感覺無法被解釋
- 人們做出相似夢境

--------------------------------------------------

### 3. 神話記憶（Mythic Memory）

- 不可直接觀測
- 影響世界結構
- 決定「什麼可以被理解」

例：
- 某些概念永遠無法完全消失
- 某些故事會自動被重述
- 某些存在必然被神話化

--------------------------------------------------

## 三、記憶生成規則

任何「被理解過的事件」都會產生三種殘留：

### 1. 事實殘留
物理結果（遺跡、痕跡）

### 2. 敘事殘留
故事版本（傳說、記錄）

### 3. 理解殘留
紫氣（未被完全解釋的部分）

--------------------------------------------------

## 四、記憶競爭機制（Memory Competition）

不同版本記憶會競爭：

### 勝出條件：

- 被更多存在重複理解
- 與當前世界穩定結構一致
- 擁有更高神話慣性

結果：

- 勝出版本成為「歷史」
- 失敗版本成為「傳說」或「禁忌」

--------------------------------------------------

## 五、遺失記憶規則（Lost Memory Rule）

記憶不會消失，而是：

### 1. 壓縮
變成符號、故事片段

### 2. 偏移
轉移到不同實相版本

### 3. 神話化
變成不可驗證的傳說

--------------------------------------------------

## 六、巨物記憶現象（Colossal Remnant Phenomenon）

當某個存在過於龐大或古老：

- 無法被完整理解
- 無法被完整記錄
- 無法被單一敘事承載

會產生：

### 「巨物化記憶」

例：

- 山脈被視為神體殘骸
- 化石被視為失敗世界版本
- 遺跡被視為理解崩塌後的殘留結構

--------------------------------------------------

## 七、恐龍級遺失（Extinction-Level Memory Loss）

當一個理解版本完全消失時：

- 物理痕跡仍可能存在
- 但其「被理解方式」消失
- 只剩可推測的碎片

結果：

> 世界仍在，但理解它的方式已滅絕

--------------------------------------------------

## 八、神話自我修補機制

當記憶缺口過大：

世界會自動補上：

- 合理化故事
- 傳說版本
- 宗教化敘述

使其重新可被理解

--------------------------------------------------

## 九、記憶與紫氣關係

紫氣是記憶的「不可消除層」：

- 事實可以被改寫
- 敘事可以被覆蓋
- 但紫氣仍保留曾經的理解痕跡

因此：

> 所有歷史都是疊層，而非替換

--------------------------------------------------

## 十、敘事意義

本系統確保：

- 世界不會真正失憶
- 也不會只有單一歷史
- 但仍能維持可敘述性

--------------------------------------------------

## 十一、設計意圖（作者備註）

本系統的目的不是保存真相，而是：

讓所有被理解過的事物，都以不同形式繼續存在。

世界不會忘記。

只是會改寫「如何記得」。
--------------------------------------------------
