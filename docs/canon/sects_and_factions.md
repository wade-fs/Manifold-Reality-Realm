# 天下教派與宗門設定總覽

> 本文件統整了《修仙志怪錄》中出現的所有教派、宗門與勢力。
> 這些宗門在故事中往往不是提供神功秘法的背景板，而是作為「巨大的規則系統」與「人的異化」來呈現。

---

## 一、 四大源流（天下修行根基）

天下修行源流大致分為四教，代表了四種面對「大道與志怪」的根本態度。

### 1. 遊玄教
**核心理念**：大道在山海天地之間，須遊歷玄境、遍觀萬象，方能明心見性。
**教風**：行走天下、探究世間局部規則、降妖破障。
**戰鬥與法寶風格**：典型「蜀山式」飛劍與行道風格。弟子多駕馭飛劍，擅長御劍穿梭。法寶多以竹木飛劍、五行靈藤、行道符水為主。

**主角所屬**：陳進出身遊玄教，為門中「少年天才」，被任命為**行道長老**，兼任**傳功院教導**。

**主要子派**：
*   **履玄派**：重視足跡與山海異象，擅長探秘境、尋天材地寶。風格：雲遊四海、飛劍開路。
*   **化生派**：以人情因果為道，擅長調停、結緣。風格：隨緣化生、廣結善緣。
*   **入世玄心派（陳進所屬）**：大道在紅塵眾生之中，須入世歷劫、混跡凡塵。常以茶館主人、遊方郎中等身份行走，以「問」之道點化眾生，降伏心魔怪獸。

### 2. 夢玄教
**核心理念**：現世如大夢，夢境方為更深之真。夢與實相互滲透，破除幻障即見本心。
**戰鬥與法寶風格**：典型「西遊式」幻術與規約法寶。不尚物理碰撞，擅長幻術、夢境投影與因果鏡射。法寶如「三生照妖鏡」、「太陰夢影扇」。
**子派**：入夢真境派、鏡玄夢影派、醒夢還源派。
**正面代表人物（規劃中）**：一位真正溫柔的夢玄教修士。他會讓讀者知道「夢」不是虛假或逃避，而是另一種理解他人痛苦與執念的方式。

### 3. 觀虛教
**核心理念**：不干涉則不增障，純以虛靜之心觀照天地萬象，如實映現。
**戰鬥與法寶風格**：典型「陣法守禦與規則法規」風格。擅長防禦結界、元素生剋與規則偏移。法寶如「五行降魔杵」、「乾坤避火罩」。
**子派**：靜觀玄照派、無跡照影派、返虛還真派。
**正面代表人物（規劃中）**：一位真正值得敬佩的觀虛教修士。他明知該介入，卻能展現出極大的克制，純以虛靜之心觀照，只在最後一刻出手。他讓讀者理解「觀」不是冷漠，而是承擔不干涉的重量。

### 4. 忘道教
**核心理念**：知見越多，障越深。唯有忘卻執著，方能返本還源、與道合一。
**戰鬥與法寶風格**：典型「野性馭獸與混亂符水」風格。崇尚反叛與野性自然，弟子行事瘋癲莫測、不拘禮法。
**子派**：忘跡無心派、無門狂玄派、還源忘牛派。
**正面代表人物（規劃中）**：一位外表瘋瘋癲癲的老修士。他總能在眾人深陷執著與規則時，一語點醒夢中人。他讓讀者看到「忘」不是逃避現實，而是真正放下執念的大智慧。

## 二、 地方宗門與特殊勢力（隨主線出現）

這些宗門多半是為了「特定目的」或「因果」而建立的系統，隨著時間推移，往往異化成了維持自身運作的官僚機構。

### 1. 慶雲宗本宗與附屬清妖所
**初次相關出現**：第16卷（除妖之後）
**特徵**：一個擁有嚴密記錄與執行體系的大型宗門網路，但本宗與附屬所的修行源流與現狀截然不同。
*   **本宗（歸屬：觀虛教 — 無跡照影派）**：高層（如梁雲嶼）掌握著全天下行道修士的路線與狀態記錄，他們不輕易干涉，只做「內部參考」。態度冷靜、克制、平等。記錄、觀察、不干涉——這是觀虛教「照而不留痕」最純粹的現實形態。
*   **附屬宗門清妖所（歸屬：遊玄教 — 履玄派，已異化）**：如方霖所在的宗門。他們的任務是「無止境地執行山谷清妖作業」。無論明年會不會來更多更強的妖，他們的應對只有一個：繼續除。這把遊玄教「降妖破障、因地制宜」的精神抽乾，變成了沒有靈魂的機械任務，是履玄派失去靈魂之後的殭屍版本，與本宗的「無跡」形成極大的荒謬對比。

### 2. 聚義堂（妖族宗門）
**初次出現**：第23卷（妖的宗門）
**歸屬**：妖族自立宗門，不歸屬四大教。
**特徵**：由妖怪組成的極度官僚化組織。
*   **起源**：三百年前由三個妖創立，初衷只是為了「一起出去劫道，分贓方便」。
*   **現狀**：發展成四百人的龐大組織。有層級、有規章、有「申請晉升程序」。原本的打劫變成了無盡的公文審批（例如為了噴火要填《消防安全承諾書》）。
*   **核心人物**：霜嬰（三百歲的文書總管）。
*   **隱喻**：一面黑色幽默的鏡子——用這種荒誕，映照人類宗門同樣走向官僚異化的過程。陳進看見這一切時，心裡閃過一個念頭：「這和孟璋的宗門，其實差不了多少。」

### 3. 小恆的宗門
**初次出現**：第三幕第34卷
**歸屬**：忘道教 — 還源忘牛派
**特徵**：這是一個罕見地沒有墮入異化、仍然保持著修行本質的宗門。
*   宗門師長對徒弟的教育非常特別，教導弟子「走得越多，會知道的越少」——這幾乎就是忘道教核心理念「知見越多，障越深」的白話翻譯。
*   在遇到無法決定的問題時，會派弟子去問陳進，即便知道陳進只會給出「不知道」，他們也認為「知道他沒有意見，比有意見更有用」。放下執有答案的衝動，才能真正看見，這是還源忘牛派最純粹的面貌。

### 4. 孟璋的宗門
**初次出現**：第一幕第3卷
**歸屬**：觀虛教 — 靜觀玄照派（已異化）
**特徵**：一個被過度「算計」與「規矩」綁架的宗門。
*   彬彬有禮、講究規矩、態度冰冷、不干涉原則被奉為圭臬——這是觀虛教的基本樣貌。
*   但這個宗門的問題在於，他們把「不干涉」從「不增業障」扭曲成了「不干涉遺址」，把觀照的對象從「當下天地萬象」換成了「過去的傳承文物」。為了取回死去的師兄留下的一塊玉簡，嚴格封鎖區域，活生生的千嶂嶺孩子反而比玉簡更不重要。這是觀虛教墮入所知障的典型樣態。
