# 《修仙志怪錄》世界觀聖經

> 全書情節皆圍繞「障」與「謎」展開，而非單純善惡對立。
> 
> 所謂障，不是罪，不是懲罰，也不是邪惡，而是一切遮蔽心性、讓人深陷執著的無形巨網。
> 
> 人人有障，妖有障，仙有障，神亦有障。
> 
> 志怪，不是為了消滅怪，而是為了在怪異中看見眾生。
> 
> 世界不是等待被征服，也不是等待被理解，而是等待被經歷與叩問。

---

## 一、核心世界觀定位

這是一部**修仙志怪小說**，融合四大中國古典敘事傳統：

| 層次 | 來源 | 功能 |
|:---|:---|:---|
| 素材層 | 《山海經》 | 地域孕育萬象；地理、異獸、異象、禁忌構成舞台 |
| 規則層 | 《封神演義》 | 力量由因果機緣決定，非數值堆疊 |
| 表現層 | 《蜀山劍俠傳》| 戰鬥重奇觀、克制、局勢；勝負非強弱對比 |
| 流程層 | 《西遊記》 | 章回單元式旅程；每段冒險皆是對荒誕世界的經歷 |

> **一句話定位**：主角不是帶著答案下山的聖人，而是在旅途中不斷遭遇未知，與讀者一起發問的旅人。

## 二、世界基本規則

### 2.1 大地理原則
- 世界**極廣**，溝通效率低，資源掌控以「鄰近地域」為主。
- 各地有自己的生靈、異象、禁忌與局部規則；沒有一條放諸四海皆準的法則。
- 秘境有各自獨立的世界觀：火域、冰原、水界、不毛之地……

### 2.2 修仙界與凡人界的邊界
- **修行界原則上不涉入凡人界**。每次介入皆累積強大的因果耦合，後患深遠。
- 話語權的取得方式：**把敵人變自己人；無法化敵為友，才消滅；且因群攻風險，必須極度克制。**

### 2.3 局部規則原則（最重要）
- 不是單一規則，而是**一堆局部規則並存**。
- 同一個能力，在不同區域表現不同。
- 同一個行為，在不同地區產生的因果不同。
- 因此：**強者不是永遠最強**；這是一個無法被完全「看透」或「總結」的世界。

## 三、修行體系

### 3.1 修為境界（由低至高）

```
旋照（入門）→ 開光 → 融合（凝氣、練氣）→ 心動 → 靈寂（= 築基）
→ 結丹 → 元嬰（★ 極重要分水嶺）→ 出竅 → 化神 → 分神 → 合體
→ 煉虛 → 問鼎 → 寂滅 → 真仙（天仙）
```

**使用原則**：
- 低境界配角，不必詳述境界等級。
- 術法無強弱，**只有因地制宜**。強弱表現在：位格 ＋ 神通 ＋ 法寶 ＋ 機緣 ＋ 因果。
- 渡劫極為兇險，除雷劫外，存在**「人間劫」**。

### 3.2 大道體系（去除說教的修行觀）

> 修行不是追求一個最終的標準答案，而是在無數的「不知道」中學會與世界共存。

在故事中，修行障不以佛理說教呈現，而是**直接具象化為取經路上的「心魔怪獸」、「詭異現象」或「地方天災」**。
主角在旅途中的成長，不是從「尋找真理」到「獲得真理」，而是從「試圖理解一切」轉變為「接受世界的荒誕，不再急於給予萬物定義」。

## 四、四層敘事架構

本書如同一座古老的山海迷宮，由四層相互交織的敘事構成。每深入一層，世界便多一重真相，也多一重迷霧。

**素材層：《山海經》式的地理**
山海經中的異獸與奇物，最初只在夢中顯形，漸漸溢出，化為真實的秘境。它們不是背景，而是**活的**——它們遵循著令人毛骨悚然的、人類無法完全理解的局部邏輯。

**規則層：《封神演義》式的因果**
力量從來不是數值，而是**因果與機緣的流轉**。把敵人變成自己人，比殺死他更難。

**表現層：《蜀山劍俠傳》式的奇觀**
戰鬥從不只是比拼境界，而是**局勢、時機、地利、人和與因果的共振**。一劍可裂山，也可能在另一處規則下連樹葉都切不動。

**流程層：《西遊記》式的章回旅程**
故事以單元志怪展開，主角是一個不斷行走的旅人。每一次相遇，都在留下無解的問題，而不是給出正確的答案。

---

## 五、夢境層
夢不是預知，也不是幻覺。
它是**另一層同樣真實的可能性**，是現實被扭曲後的倒影。
夢境與現實的邊界極為模糊，常無明顯標記便相互滲透。

---

## 六、情節核心驅動：「這件事找誰解決？」

**故事推進邏輯不是「變更強」，而是「找到對的線索與人」**：
- 找對的人、法寶、關係（人情即因果）。
- 找到妖怪或難題的**真正來歷**。
- 主角的成長，是學會辨識規則、累積歷程，而非把自己堆成無敵。

---

## 七、志怪隱喻寫作準則

本書取法魏晉至明清志怪傳統：
1. **以「物」喻「人」**：妖、鬼、精是人間慾望的投射。
2. **以「因果」喻「命運」**：果報有模糊性與複雜性；善惡難辨。
3. **留白與多義性**：避免直白說教，用最少的筆墨點到即止。
4. **含蓄性**：用「怪」製造安全距離，不正面說破。

### 關於「西遊八步」的敘事節奏（極重要）：

這個八步流程（到地方 → 被誤會 → 被追殺 → 找不到人 → 妖怪不是妖怪 → 神仙搞事 → 唐僧亂發善心 → 搬救兵與結束）是**一個完整單元（例如 8 到 10 章）**的結構，**絕對不是**要求在「單一章節」內走完！

- **單一章節不需要完整**：不要在第一章就想著破霧或解決問題。
- **單一章節可以是非常失敗的**：主角可以被警告、不聽勸、然後直接被怪異吞噬（例如拖入夢境），章節就此戛然而止。
- **章節結尾必須是「下回分解」**：保留強烈的懸念，讓讀者迫不及待想看下一章。

| 誤區 | 正確做法 |
|:---|:---|
| 主角帶著答案下山，動輒說教 | 主角常常感到困惑，回答「不知道」 |
| 使用「觀測」、「理解結構」等現代詞彙 | 使用古典志怪語境，描繪荒誕現象本身 |
| 章節結尾「大徹大悟、真相大白」 | 危機解除，但留下毛骨悚然或無解的餘韻 |
| 把破障寫成一次完成 | 障會反覆出現，修行是終身歷程 |
| 提早為法寶或現象命名 | 延遲命名，一開始只是「破鏡子」、「怪霧」 |
| 把修行障寫成純粹的反派 | 修行障是扭曲規則下的悲劇與執念 |
