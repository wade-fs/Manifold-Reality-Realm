# 《修仙志怪錄》世界觀聖經

> 本文件為寫作的最高綱要，所有章節均應以此為依據。

---

## 一、核心世界觀定位

這是一部**修仙志怪小說**，融合四大中國古典敘事傳統：

| 層次 | 來源 | 功能 |
|:---|:---|:---|
| 素材層 | 《山海經》 | 地域孕育萬象；地理、異獸、異象、禁忌構成舞台 |
| 規則層 | 《封神演義》 | 力量由因果機緣決定，非數值堆疊 |
| 表現層 | 《蜀山劍俠傳》| 戰鬥重奇觀、克制、局勢；勝負非強弱對比 |
| 流程層 | 《西遊記》 | 章回單元式旅程；每段冒險皆是對世界的探索 |

> **一句話定位**：主角不是越來越強的英雄，而是越來越懂這個世界的旅人。

## 二、世界基本規則

### 2.1 大地理原則
- 世界**極廣**，溝通效率低，資源掌控以「鄰近地域」為主。
- 各地有自己的生靈、異象、禁忌與局部規則；沒有一條放諸四海皆準的法則。
- 山海經的珍稀材料**不會到處都有**，稀缺性催生探險。
- 秘境（異界入口，類似黑洞連通的異世界）有各自獨立的世界觀：火域、冰原、水界、不毛之地……

### 2.2 修仙界與凡人界的邊界
- **修行界原則上不涉入凡人界**。每次介入皆累積強大的因果耦合，後患深遠。
- 修仙界不是弱肉強食——資源夠用，不夠就去探險開發新域，而非互相吞噬。
- 話語權的取得方式：**把敵人變自己人；無法化敵為友，才消滅；且因群攻風險，必須極度克制。**

### 2.3 局部規則原則（最重要）
在秘境與夢境中：
- 不是單一規則，而是**一堆局部規則並存**。
- 同一個能力，在不同區域表現不同。
- 同一個神，在不同地區的權限不同。
- 同一個行為，在不同地區產生的因果不同。
- 因此：**強者不是永遠最強**；永遠需要不斷累積歷程，學會辨識並運用當地規則。

## 三、修行體系

### 3.1 修為境界（由低至高）

```
旋照（入門）→ 開光 → 融合（凝氣、練氣）→ 心動 → 靈寂（= 築基）
→ 結丹 → 元嬰（★ 極重要分水嶺）→ 出竅 → 化神 → 分神 → 合體
→ 煉虛 → 問鼎 → 寂滅 → 真仙（天仙）
```

**使用原則**：
- 低境界配角，不必詳述境界等級。
- 境界不是重點；**道心 + 靈氣積累 + 感悟天道** 才是核心。
- 術法無強弱，**只有因地制宜**。強弱表現在：位格 ＋ 神通 ＋ 法寶 ＋ 機緣 ＋ 因果。
- 渡劫極為兇險，需準備充足（法寶、丹藥、心境）。
- 除雷劫外，存在**「人間劫」**（時間漫長、如心劫般消磨意志的試煉期）。
- 後期設定有仙界（地仙、天仙）與神界。

### 3.2 大道體系（十牛圖）

主角的心境成長對應十牛圖次第：

| 階段 | 圖名 | 隱喻意涵 |
|:---|:---|:---|
| 1 | 尋牛 | 迷失，開始尋找自我與天道 |
| 2 | 見跡 | 發現線索，初窺規律 |
| 3 | 得牛 | 掌握某一道之初步 |
| 4 | 牧牛 | 馴服，磨合自身與道的關係 |
| 5 | 騎牛歸家 | 與道合一，自然而然 |
| 6 | 忘牛存人 | 道已內化，不再需要刻意執持 |
| 7 | 人牛俱忘 | 超越對立，空性現前 |
| 8 | 返本還源 | 回歸世界，以平常心觀萬物 |
| 9 | 入廛垂手 | 入世度人，無為而無不為 |

## 四、四層敘事架構

### 4.1 素材層：《山海經》式地理
- 地域塑造世界，而非地圖劃分；每一方水土有自己的生靈、禁忌與真理。
- 山海經的怪獸素材**最初只出現在夢境**，慢慢溢出到現實中，以**秘境**的形式存在。

### 4.2 規則層：《封神》式因果
- 人人一開始就有功法；能力不論出身，只論行止與機緣。
- **切勿陷入純因果流**；因果只是彼此牽涉的因素之一，不是故事主驅動。

### 4.3 表現層：《蜀山》式戰鬥
- 帶入山海經的地圖素材與蜀山的奇觀感。
- 勝負由**局勢**生成：天時、地利、人和、因緣。
- 法術與法寶是工具，不是能力榜的數字。

### 4.4 流程層：《西遊》式章回
- 單元結構；門派頒布任務（非順序，清單型）。
- 典型任務類型：
  - 門派大比（門內晉升、選材）
  - 區域話語權競爭
  - 門派競合壯大
  - 秘境掌控權爭奪（無法獨吞，是一種共享掌控）
  - 修行法門的保護或消滅（知識即權力）
- **主角只是世界改變的見證者與參與者**，不必最強，也不必踩遍所有人。

### 4.5 夢境層
- 夢不是預知，而是**現實的扭曲映射與另一種可能性**。
- 夢境漸漸影響現實，這是一種因果牽扯，而非能力的具現化。
- 夢境與現實邊界模糊：「人生本如大夢」、「現實亦是某種集體幻覺」。

### 4.6 隱喻層
- 借「怪」說「人」，借「異」論「世」。
- 隱喻人性原罪（七宗罪）、人性黑暗面，解方援引卡內基式人際智慧與佛教修行觀。
- 以小見大；極短篇幅承載極大信息量。

## 五、情節核心驅動：「這件事找誰解決？」

**故事推進邏輯不是「變更強」，而是「找到對的資源」**：

- 找對的人。
- 找對的法寶。
- 找對的關係（人情即因果）。
- 找到妖怪或難題的**真正來歷**（知道根源才能解決）。
- 跨越每一個地方，都必須**重新理解一套知識**。

> 類比：孫悟空不是最強，但他能找最多助手。主角的成長，是學會辨識規則、找到關鍵人脈、累積歷程，而非把自己堆成無敵。

## 六、七宗罪嵌入體系

**全書情節嵌入七宗罪的輻射結構**：

| 罪行 | 對立美德 | 嵌入方式 |
|:---|:---|:---|
| 驕傲 | 謙遜 | 門派高層或掌門的核心性格缺陷 |
| 嫉妒 | 寬仁 / 慷慨 | 同門之間、門派之間的競爭動機 |
| 憤怒 | 忍耐 / 溫和 | 主角或同門的情緒失控弧線 |
| 怠惰 | 勤勉 | 旅程中遇到的反面案例 |
| 貪婪 | 慷慨 | 區域話語權與秘境爭奪的根源 |
| 暴食 | 節制 | 沉溺修行捷徑或靈材的執念 |
| 色慾 | 純潔 / 貞潔 | 感情線、誘惑機關（秘境、修行岐路）|

**使用原則**：
- 不要把七宗罪集中在一個人身上；分散至不同角色與不同章回。
- 不同夢境受不同罪的影響染色。
- 主角是見證者，**可以是救贖者，但不一定是**。

## 七、角色設計模式

### 7.1 核心主角
- **定義**：所有因果最終都能追溯到他；是世界的旅人，不是世界的主人。
- **五種可積累資產**：

| 資產 | 說明 |
|:---|:---|
| 見聞 | 走過各地的知識積累；七宗罪的目擊者，有時是救贖者 |
| 人情 | 因果網絡，有正增上緣（助力）也有逆增上緣（業障） |
| 身分 | 隨章回獲得宗門（含他派）的認可，逐步提升地位 |
| 本事 | 法術、法寶、技巧的積累 |
| 使命 | 可以是完成大一統，或其他長線目標 |

### 7.2 軌道主角（章回主角）
- 每個章回有自己的「軌道主角」主持該回，有允入 / 允出條件。
- 後續章回可再出現，但須**逐漸淡化**，避免喧賓奪主。
- 主角不會因進入而破壞軌道世界，**可以拯救，但不一定**。

## 八、志怪隱喻寫作準則

本書取法魏晉至明清志怪傳統（《搜神記》、《聊齋誌異》、《閱微草堂筆記》等）：

### 8.1 核心手法
1. **以「物」喻「人」**：妖、鬼、狐、精是人間性格、慾望的投射，而非單純的反派。
2. **以「怪異事件」喻「社會現實」**：借荒誕情節折射人性、關係、權力結構。
3. **以「因果輪迴」喻「道德與命運」**：果報有模糊性與複雜性；善惡難辨，人生無常。
4. **以「夢」作雙層空間**：夢是現實的扭曲映射，或另一種可能性；夢與現實相互滲透。
5. **留白與多義性**：同一意象可有不同解讀；避免直白說教，用最少的筆墨點到即止。

### 8.2 文字風格
- **含蓄性**：用「怪」製造安全距離，不正面衝突，保留詮釋空間。
- **多義性**：同一意象可喻不同對象（如狐可喻女性、隱士、慾望）。
- **世俗性與超越性並存**：接地氣（市井人情）且帶哲思（虛實、命運、人性）。
- **以小見大**：極短篇幅，承載極大信息量。

## 九、常見誤區（禁止事項）

| 誤區 | 正確做法 |
|:---|:---|
| 以境界高低決定勝負 | 以局勢、因地制宜、機緣決定勝負 |
| 主角獨力解決一切 | 主角找對的人、法寶、關係來解決 |
| 故事靠主線強行驅動 | 靠旅程、章回任務、局部世界自然展開 |
| 因果流過於機械（報應必現） | 因果有遲滯性、模糊性，保留複雜性 |
| 怪獸只是障礙 | 怪獸有來歷、有隱喻；找出根源才是解題關鍵 |
| 主角進入軌道即主導一切 | 主角是旅人；軌道世界有自己的邏輯 |
| 七宗罪集中於一人 | 分散至不同角色與章回，避免臉譜化 |
