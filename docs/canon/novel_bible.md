# 《修仙志怪錄》世界觀聖經

> 全書情節皆圍繞「修行障」展開，而非單純善惡對立。
> 
> 所謂障，不是罪，不是懲罰，也不是邪惡，而是一切遮蔽心性、妨礙如實觀照世界的執著、習氣與偏見。
> 
> 人人有障，妖有障，仙有障，神亦有障。
> 
> 修行，不在消滅他人，而在照見自己。
> 
> 見怪 → 見人 → 見己 → 破障。
>
> 世界不是等待被征服，而是等待被理解。

---

## 一、核心世界觀定位

這是一部**修仙志怪小說**，融合四大中國古典敘事傳統：

| 層次 | 來源 | 功能 |
|:---|:---|:---|
| 素材層 | 《山海經》 | 地域孕育萬象；地理、異獸、異象、禁忌構成舞台 |
| 規則層 | 《封神演義》 | 力量由因果機緣決定，非數值堆疊 |
| 表現層 | 《蜀山劍俠傳》| 戰鬥重奇觀、克制、局勢；勝負非強弱對比 |
| 流程層 | 《西遊記》 | 章回單元式旅程；每段冒險皆是對世界的探索 |

> **一句話定位**：主角不是越來越強的英雄，而是在旅途中不斷破除自身與眾生之障，逐漸理解天地萬物的旅人。

## 二、世界基本規則

### 2.1 大地理原則
- 世界**極廣**，溝通效率低，資源掌控以「鄰近地域」為主。
- 各地有自己的生靈、異象、禁忌與局部規則；沒有一條放諸四海皆準的法則。
- 山海經的珍稀材料**不會到處都有**，稀缺性催生探險。
- 秘境（異界入口，類似黑洞連通的異世界）有各自獨立的世界觀：火域、冰原、水界、不毛之地……

### 2.2 修仙界與凡人界的邊界
- **修行界原則上不涉入凡人界**。每次介入皆累積強大的因果耦合，後患深遠。
- 修仙界不是弱肉強食——資源夠用，不夠就去探險開發新域，而非互相吞噬。
- 話語權的取得方式：**把敵人變自己人；無法化敵為友，才消滅；且因群攻風險，必須極度克制。**

### 2.3 局部規則原則（最重要）
在秘境與夢境中：
- 不是單一規則，而是**一堆局部規則並存**。
- 同一個能力，在不同區域表現不同。
- 同一個神，在不同地區的權限不同。
- 同一個行為，在不同地區產生的因果不同。
- 因此：**強者不是永遠最強**；永遠需要不斷累積歷程，學會辨識並運用當地規則。

## 三、修行體系

### 3.1 修為境界（由低至高）

```
旋照（入門）→ 開光 → 融合（凝氣、練氣）→ 心動 → 靈寂（= 築基）
→ 結丹 → 元嬰（★ 極重要分水嶺）→ 出竅 → 化神 → 分神 → 合體
→ 煉虛 → 問鼎 → 寂滅 → 真仙（天仙）
```

**使用原則**：
- 低境界配角，不必詳述境界等級。
- 境界不是重點；**道心 + 靈氣積累 + 感悟天道** 才是核心。
- 術法無強弱，**只有因地制宜**。強弱表現在：位格 ＋ 神通 ＋ 法寶 ＋ 機緣 ＋ 因果。
- 渡劫極為兇險，需準備充足（法寶、丹藥、心境）。
- 除雷劫外，存在**「人間劫」**（時間漫長、如心劫般消磨意志的試煉期）。
- 後期設定有仙界（地仙、天仙）與神界。

### 3.2 大道體系（十牛圖）

> 修行不是追求大道，而是不斷破除遮蔽大道的障礙。
> 
> 大道未曾遠離，遠離的是自己的心。

主角的心境成長對應十牛圖次第：

| 階段 | 圖名     | 隱喻意涵                   | 修行障                |
|:---- |:-------- |:-------------------------- | :---------------------|
| 1    | 尋牛     | 迷失，開始尋找自我與天道   | 就是開始知道自己有障  |
| 2    | 見跡     | 發現線索，初窺規律         | 就是開始知道障在哪裡  |
| 3    | 見牛     | 隱約看見大道的真容         | 就是逐漸看清障的本質  |
| 4    | 得牛     | 掌握某一道之初步           | 就是第一次破障        |
| 5    | 牧牛     | 馴服，磨合自身與道的關係   | 就是反覆破同一個障    |
| 6    | 騎牛歸家 | 與道合一，自然而然         |                       |
| 7    | 忘牛存人 | 道已內化，不再需要刻意執持 | 就是不再被障控制      |
| 8    | 人牛俱忘 | 超越對立，空性現前         |                       |
| 9    | 返本還源 | 回歸世界，以平常心觀萬物   |                       |
| 10   | 入廛垂手 | 入世度人，無為而無不為     |                       |

使用原則：

	- **「青牛」的實體化神話設定**：十牛圖雖然隱喻著心境轉變，但在故事世界中，**「青牛」是一頭由虛空法則凝聚的太古神獸（大道力量的具象化）**。主角在旅途中的「尋牛、得牛、牧牛」等階段，不只是抽象的開悟，更是一場場與這頭野性難馴的法则神獸正面交鋒、降伏、磨合並最終「騎牛歸家」的精彩仙俠冒險（類似孫悟空在取經路上馴化心猿的歷程）。
	- **「修行障」的具象化妖魔設定**：所謂「修行障」在正文中不以枯燥的佛理說教呈現，而是**直接具象化為取經路上的「心魔怪獸」、「邪門陣法」或「地方天災」**（例如：貪障會化身為掠奪生機的晶石怪物；瞋障會化身為引起火山爆發的怒火地靈；慢障則表現為自封神仙的割據散修）。主角與其鬥智鬥勇、尋求關係、借用法寶來破除這些阻礙的過程，就是破障修行的具體戰鬥展現。
	- 障礙雖有十萬八千，但在情節上聚焦於：散亂、退轉、所知障、所緣障、習氣反撲，以及經典的貪、瞋、癡、慢、疑。降妖即是破障，伏魔即是照己。

### **四、四層敘事架構**

本書如同一座古老的山海迷宮，由四層相互交織的敘事構成。每深入一層，世界便多一重真相，也多一重迷霧。

**素材層：《山海經》式的地理**

世界極其廣大，溝通艱難。每一片水土、每一座山峰、每一條河流，都有自己孕育的生靈、禁忌與隱秘規則。山海經中的異獸與奇物，最初只在夢中顯形，漸漸溢出，化為真實的秘境。它們不是背景，而是**活的**——它們在觀測者到來之前，可能還未完全「成型」。

**規則層：《封神演義》式的因果**

力量從來不是數值，而是**因果與機緣的流轉**。人人皆有功法，卻無人能獨占大道。真正的強弱，取決於你能否理解眼前這方天地的「真正規矩」。把敵人變成自己人，比殺死他更難，也更接近大道。

**表現層：《蜀山劍俠傳》式的奇觀**

戰鬥從不只是比拼境界，而是**局勢、時機、地利、人和與因果的共振**。一劍可裂山，也可能在另一處規則下連樹葉都切不動。勝負往往在瞬息之間，由「理解的深淺」決定。

**流程層：《西遊記》式的章回旅程**

故事以單元志怪展開，每一章都是一次獨立的「遇怪」。主角不是救世主，而是一個不斷行走的旅人。他走過的不是地圖，而是一層又一層被不同理解穩定的世界。每一次相遇，都在問同一個問題：

這件事，真正困住的是誰？

---

**同時建議在「夢境層」加強**：

### **4.5 夢境層**

夢不是預知，也不是幻覺。

它是**另一層同樣真實的可能性**，是現實被扭曲後的倒影，也是尚未被穩定下來的世界版本。

夢境與現實的邊界極為模糊，常無明顯標記便相互滲透。有時，你以為身在現實，卻已踏入他人的夢；有時，你在夢中醒來，卻發現枕邊多了一片只屬於夢境的鱗片。

## 五、情節核心驅動：「這件事找誰解決？」與 「這件事，真正困住的是誰？」

**故事推進邏輯不是「變更強」，而是「找到對的資源」**：

- 找對的人。
- 找對的法寶。
- 找對的關係（人情即因果）。
- 找到妖怪或難題的**真正來歷**（知道根源才能解決）。
- 跨越每一個地方，都必須**重新理解一套知識**。

> 類比：孫悟空不是最強，但他能找最多助手。主角的成長，是學會辨識規則、找到關鍵人脈、累積歷程，而非把自己堆成無敵。

## 六、修行障體系

然後內容改成：

全書情節皆圍繞「障」展開，而非單純善惡對立。

| 修行障 | 本質    | 劇情呈現          |
| --- | ----- | ------------- |
| 貪   | 求不得   | 執著機緣、法寶、權勢    |
| 瞋   | 放不過   | 復仇、怨恨、殺心      |
| 癡   | 看不明   | 幻境、誤判、自欺      |
| 慢   | 放不下自己 | 自負、門戶之見       |
| 疑   | 不信、不決 | 猶豫、猜忌、失去道心    |
| 見知障 | 執己見   | 自認理解天道，不願改變   |
| 所知障 | 執知識   | 學問越深，反被學問束縛   |
| 所緣障 | 執境界   | 執著師門、情愛、身份、道統 |
| 散亂  | 心不定   | 修行停滯、失去方向     |
| 退轉  | 志不堅   | 放棄初心、畏難而退     |
| 習氣  | 業力慣性  | 明知不可仍反覆犯錯     |


最後補一句：

每個章回不必同時包含所有障，而是聚焦於一種或數種障。主角不一定破障，也可能只是看見障；真正的修行，是一次次照見自己與眾生的迷失。

**使用原則**：
- 不要把所有修行障集中在一個人身上；分散至不同角色與不同章回。
- 不同夢境受不同罪的影響染色。
- 主角是見證者，**可以是救贖者，但不一定是**。

## 七、角色設計模式

### 7.1 核心主角
- **定義**：所有因果最終都能追溯到他；是世界的旅人，不是世界的主人。

    主角真正累積的，不只是見聞、人情與本事，更重要的是一次次破障的歷程。
    每一次旅程，都使主角對世界的理解更深，也讓他逐漸放下過去的執著；因此，修行不是力量的累積，而是觀看方式的轉變。

- **五種可積累資產**：

| 資產 | 說明 |
|:---|:---|
| 見聞 | 走過各地的知識積累；修行障的目擊者，有時是救贖者 |
| 人情 | 因果網絡，有正增上緣（助力）也有逆增上緣（業障） |
| 身分 | 隨章回獲得宗門（含他派）的認可，逐步提升地位 |
| 本事 | 法術、法寶、技巧的積累 |
| 心境 | 一次次照見與破除修行障，觀看世界的方式逐漸改變 |

### 7.2 軌道主角（章回主角）
- 每個章回有自己的「軌道主角」主持該回，有允入 / 允出條件。
- 後續章回可再出現，但須**逐漸淡化**，避免喧賓奪主。
- 主角不會因進入而破壞軌道世界，**可以拯救，但不一定**。

## 八、志怪隱喻寫作準則

本書取法魏晉至明清志怪傳統（《搜神記》、《聊齋誌異》、《閱微草堂筆記》等）：

### 8.1 核心手法
1. **以「物」喻「人」**：妖、鬼、狐、精是人間性格、慾望的投射，而非單純的反派。
2. **以「怪異事件」喻「社會現實」**：借荒誕情節折射人性、關係、權力結構。
3. **以「因果輪迴」喻「道德與命運」**：果報有模糊性與複雜性；善惡難辨，人生無常。
4. **以「夢」作雙層空間**：夢是現實的扭曲映射，或另一種可能性；夢與現實相互滲透。
5. **留白與多義性**：同一意象可有不同解讀；避免直白說教，用最少的筆墨點到即止。

### 8.2 文字風格
- **含蓄性**：用「怪」製造安全距離，不正面衝突，保留詮釋空間。
- **多義性**：同一意象可喻不同對象（如狐可喻女性、隱士、慾望）。
- **世俗性與超越性並存**：接地氣（市井人情）且帶哲思（虛實、命運、人性）。
- **以小見大**：極短篇幅，承載極大信息量。

### 每一個章回，都應從一個值得追問的問題開始，而不是從一件怪事開始。怪只是表象，問題才是故事真正的核心。
### 每一個章回大致流程:
- 到地方
- →被誤會
- →被追殺
- →找不到人
- →妖怪不是妖怪
- →神仙搞事
- →唐僧亂發善心
- →悟空跑去搬救兵
- →事情才結束
### 舉例:
- 原來流程: 老蛇 ↓ 香火 ↓ 異物 ↓ 救蛇
	- 新流程: 有人一直拜 ↓ 願望一直成功 ↓ 全鎮開始瘋狂拜 ↓ 所有願望都開始成功 ↓ 事情開始失控 ↓ 原來不是神顯靈 而是每完成一個願望 就偷走另一個人的東西。
- 原來流程: 畫師畫不完 ↓ 所知障 ↓ 放下
	- 新流程: 畫師失蹤 ↓ 所有人都懷疑畫師瘋了 ↓ 有人開始模仿他的畫 ↓ 整個澤開始變成畫 ↓ 大家都以為是畫師造成 ↓ 最後才知道 畫師一直在阻止朱鳥被畫完

## 九、常見誤區（禁止事項）

| 誤區 | 正確做法 |
|:---|:---|
| 以境界高低決定勝負 | 以局勢、因地制宜、機緣決定勝負 |
| 主角獨力解決一切 | 主角找對的人、法寶、關係來解決 |
| 故事靠主線強行驅動 | 靠旅程、章回任務、局部世界自然展開 |
| 因果流過於機械（報應必現） | 因果有遲滯性、模糊性，保留複雜性 |
| 怪獸只是障礙 | 怪獸有來歷、有隱喻；找出根源才是解題關鍵 |
| 主角進入軌道即主導一切 | 主角是旅人；軌道世界有自己的邏輯 |
| 修行障集中於一人 | 分散至不同角色與章回，避免臉譜化 |
| 把修行障寫成善惡對立 | 修行障人人皆有，只是顯現方式不同 |
| 把破障寫成一次完成 | 障會反覆出現，修行是終身歷程 |

## 十、一句話寫作錨點

> 每寫一章前，先問自己：

    . 這一章，誰被什麼障困住？
    . 這個志怪映照了什麼人性？
    . 主角在這裡學會了什麼新的觀看方式？
    . 這一次，他破了什麼障？又留下了什麼新的障？
