# 人物允入條件（Character Entry Rules）

## 核心定義

本規則用來決定：

「什麼樣的存在，可以被允許進入《多重實相界》作為『可敘事角色』。」

注意：

這不是角色列表，也不是職業設定。

這是「世界允許什麼樣的人性存在」的規則。

--------------------------------------------------

## 一、存在性條件（Existence Condition）

一個角色必須至少滿足以下其中一種存在形式，才能被允許進入敘事層：

### 1. 理解驅動存在（Understanding-Driven Entity）

其行動動機不是「慾望」，而是「理解」。

特徵：

- 會因為「想理解」而旅行
- 會改變自身立場，而不是固守身份
- 不以勝利為終點，而以「看清真相」為終點

代表型角色：
陳進（原型）

--------------------------------------------------

### 2. 關係驅動存在（Relational Entity）

其存在意義來自「與他者的關係」。

特徵：

- 無法獨立定義自己
- 在不同對象面前會呈現不同形態
- 會因關係改變而改變自身本質

代表型角色：
郭芳（原型）

--------------------------------------------------

### 3. 概念化存在（Conceptual Entity）

不是「人」，而是「概念被人格化」。

特徵：

- 代表一種抽象存在（例如：慈悲、秩序、混亂）
- 其行為不完全自由，而是遵循概念邏輯
- 可能同時存在於多個實體之中

代表型：
混沌之母（原型）
山神 / 河神 / 城隍（後續可延伸）

--------------------------------------------------

## 二、敘事可變條件（Narrative Mutation Condition）

一個角色必須具備「至少一種可變性」，否則不得進入主線敘事：

### 1. 認知可變性
角色能因經驗改變「對世界的理解方式」

### 2. 身份可變性
角色可以在敘事中改變身份（人 → 神 / 妖 → 鬼 / 等）

### 3. 立場可變性
角色不固定站在「善 / 惡 / 正 / 邪」任一端

--------------------------------------------------

## 三、禁止條件（Hard Exclusion Rules）

以下類型不可作為核心敘事角色：

### 1. 絕對固定角色
- 永遠不變
- 永遠正確 / 永遠邪惡
- 沒有成長或變化空間

### 2. 純功能角色
- 只負責推劇情（例如：送寶、報信、工具人）
- 沒有自身存在問題

### 3. 單一維度角色
- 只有「強 / 弱」「善 / 惡」單一屬性

--------------------------------------------------

## 四、角色存在目的規則（Purpose Rule）

所有允入角色必須滿足：

「他存在於故事中，是為了改變世界的理解方式，而不是單純推進事件。」

--------------------------------------------------

## 五、特殊規則：紫氣適格者（Optional Rule）

若角色可與「紫氣」產生共鳴，必須滿足：

- 能承受「世界被重新理解」的結果
- 不會因失去既有認知而崩潰
- 願意接受「世界可能不是他所理解的樣子」

--------------------------------------------------

## 六、設計意圖（作者用備註）

這套規則的目的不是限制角色數量，而是：

過濾掉「不能成為神話的人」。

讓所有進入世界的存在，都具備一個共同點：

「他們的存在，本身就會改變世界如何被理解。」

--------------------------------------------------
