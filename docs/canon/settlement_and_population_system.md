# 《多重實相界》聚落與生民系統（settlement_and_population_system.md）

## 太古之理

天地既分，人煙初起。
聚落非僅是生民聚居之所，更是**天地理解的結晶**與**實相穩定的胎盤**。

生民亦非尋常血肉，而是**理解的載體**，是天地自我觀照的無數眼睛。

---

## 聚落之象

### 1. 初生聚落（單一實相之鄉）
- 特徵：生民共享單一理解，天地在此處尚未裂開
- 象徵：如第一卷開頭的山腳村落，隱於太古之霧中
- 狀態：脆弱而純樸，極易被異象驚醒

### 2. 重影聚落（多界共存之鄉）
- 特徵：同一片土地同時存在多個命運版本
- 象徵：第20章的三村重影——繁榮、荒廢、從未存在
- 狀態：如鏡面破碎又重新拼合，充滿神異與不安

### 3. 界域聚落（門派化之鄉）
- 特徵：由單一強烈理解主導，形成穩固的「小界」
- 例子：
  - **息谷**：以呼吸為律令，風與草木皆隨心而動
  - **劍宗山頭**：山石如被天刀反覆雕琢，充滿鋒芒
  - **氣宗流域**：空間如水，人如游魚
- 狀態：高度穩定，卻也因此僵化

### 4. 妖城（生生之鄉）
- 特徵：規則持續變異，建築與生靈邊界模糊
- 象徵：活的山海經，城池如巨獸般呼吸、生長、蛻變
- 狀態：充滿生機，亦充滿危險

### 5. 無名聚落（無觀之鄉）
- 特徵：無主流理解，純粹自生成
- 狀態：最終形態，天地自足，不再需要被觀測

## 生民之象

生民乃天地之眼，亦是天地的負擔。

### 主要類型
- **凡民**：持日常理解而活，最為眾多
- **覺醒者**：如陳進，能察覺天地細微的裂隙
- **執道者**：如張固執，堅守單一理解至死
- **行界者**：如陳進與青霧，能穿梭諸界而不迷失
- **承載者**：後期陳進之象，能同時容納矛盾而不崩

### 生民流轉
- **同化**：久居某界，血肉與心念皆被該界同化
- **分裂**：理解衝突過烈，一人可化為多個版本
- **消逝**：理解被天地排斥，則被地府收回，化為幽冥書頁
- **重生**：壓縮後的記憶可重新投入妖界，化為新的生民

---

## 聚落與天地的共鳴

- 聚落越大，越難維持單一理解（易生分裂）
- 聚落越小，越易孕育極致穩固的小界
- 邊緣聚落（山村、界隙、廢墟）乃新神話誕生的聖地

當聚落內的理解達到某種臨界，天地便會「認可」它，賜予更穩固的法則，如同為其加冕。
