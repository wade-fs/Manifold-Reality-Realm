# 《多重實相界》聚落與生民系統（settlement_and_population_system.md）

## 太初之語

天地既分，人煙初起。
聚落非僅是生民聚居之所，更是**天地理解的結晶**與**實相穩定的胎盤**。

生民亦非尋常血肉，而是**理解的載體**，是天地自我觀照的無數眼睛。

---

## 聚落之象（依時間線演變）

### 第一卷　霧生初界（單一實相之鄉）
- 特徵：生民共享單一理解，天地尚未明顯裂開
- 例子：陳進出身的南方山村（紫氣東來之地）
- 狀態：純樸脆弱，容易被異象驚醒

### 第二卷　界起爭鋒（界域聚落興起）
- 特徵：由強烈理解主導，形成穩固的「小界」
- 例子：
  - 太初劍宗山門：山石如被天刀雕琢
  - 玄天氣宗流域：空間如活水
  - 不滅體宗聖地：山巒如巨人之軀
- 狀態：高度穩定，但也容易僵化與衝突

### 第三卷　神話重臨（神話重影聚落）
- 特徵：同一片土地同時存在多重神話版本
- 例子：山海經異獸活動區、封神古戰場投影、西遊古徑交疊之地
- 狀態：神異而混亂，充滿不可預測的變化

### 第四卷　天庭裂變（規則戰爭聚落）
- 特徵：規則衝突激烈，聚落分裂成多個版本
- 例子：天庭正統控制的「統一城」、協調網的「多元城」、規則廢土邊緣的「漂流村」
- 狀態：動盪不安，居民常成為規則難民

### 第五卷　歸一危機（空白區與收縮聚落）
- 特徵：實相開始坍縮，聚落逐漸失去多樣性
- 例子：空白區周邊的「消逝村」、最後的穩定聚落
- 狀態：世界變得單薄而壓抑

### 第六卷　息滅重生（重構聚落）
- 特徵：世界冷啟動後出現新形態聚落
- 例子：以元規則韻律為基礎的「新生聚落」

### 第七卷　無觀歸真（自生成聚落）
- 特徵：不再需要被觀測即可穩定存在
- 例子：最終的「無觀之鄉」

## 生民之象

- **凡民**：持日常理解而活
- **覺醒者**：如陳進，能察覺天地裂隙
- **界行者**：如陳進與郭芳，能穿梭諸界
- **承載者**：後期陳進，能同時容納矛盾

**生民流轉**：
- 同化、分裂、消逝、重生
