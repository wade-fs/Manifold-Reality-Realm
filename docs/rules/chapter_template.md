# 修仙志怪錄 章節寫作模板

**章節標題格式**：`#### 第 N 章 【標題】`

---

## 標準章節結構（建議字數分配）

### 開場（300～500字）
- 時間、地點、陳進當下身體與困惑的狀態
- 細膩環境描寫（光影、氣味、聲音、空間質感）
- 毫無預警地拋出本章「無法解釋的怪異現象」
- 建立讀者毛骨悚然或荒誕的沉浸感

### 遭遇與發問段（1500～2000字）
- 陳進主動或被動捲入異常現象
- 詳細描寫他「看見」的詭異與矛盾之處
- 展現他的真實無知。他不試圖給出高深的解釋，只是發出最樸素的疑問（如「這裡哪裡有霧？」）
- 逐步累積懸疑張力，不急於給出答案

### 破局與錯位段（1500～2000字）
- 當地人或神仙給出荒謬的邏輯，或者局勢徹底失控
- 陳進不用毀滅性的法術，而是用極端反常規、世俗或近乎兒戲的方式（如吹滅一盞燈、說一句「不知道」）打破僵局
- 過程充滿錯位感，不寫長篇大論的鬥法，寫規則的荒謬

### 餘韻與留白段（400～600字）
- 危機雖然解除，但最核心的問題依然沒有標準答案
- 展現本章事件留下的怪異餘韻
- 嚴禁陳進在章末做出「大徹大悟、心中清明」的感悟，他應該比一開始帶著更深的疑問繼續上路

### 收尾與伏筆（300～400字）
- 留下下一章的自然懸念或新線索
- 保持大霧瀰漫、前路未知的氛圍

---

## 字數要求
- **單章正文字數**：至少 3500 字（純中文）
- **目標字數**：3500～5500 字（舒適範圍）

---

## 寫作提醒
- 視角嚴格使用**第三人稱有限視角**（緊跟陳進）
- **極度嚴格禁止**：出現「觀測」、「理解結構」、「本質」等帶有現代科學或過度說教意味的詞彙。
- **極度嚴格禁止**：主角講述大道理，或在心裡做長篇大論的哲學總結。
- 每章必須體現一次「遭遇怪異 → 無法理解 → 凡俗破局 → 留下更深疑問」的循環。
- 語言保持乾淨、克制、充滿志怪的「留白」。
